# AI-Powered Lead Generation System

## Project Overview
This project is developed using Salesforce CRM. It automates lead management using custom objects, validation rules, reports, dashboards, flows, and approval processes.

## Features
- Custom Object (Lead Information)
- Custom Fields
- Validation Rules
- Formula Field
- Reports
- Dashboards
- Record Triggered Flow
- Approval Process
- Email Alert

## Technologies Used
- Salesforce CRM
- Flow Builder
- Reports & Dashboards
- Approval Process
